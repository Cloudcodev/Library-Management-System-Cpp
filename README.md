Library Management System (C++)

A console-based library management system built to demonstrate C++ fundamentals, memory management, and data structures.

Features
* Add Books: detailed entry with ID, Title, Author, and ISBN.
* List Inventory: dynamically displays all books currently in memory.
* Input Validation: Robust error handling prevents crashes from invalid inputs.
* Data Structures: Uses `struct` for book entities and `vector` for dynamic storage.

Tech Stack:
* Language: C++
* Concepts: Vectors, Structs, Loops, Input Streams
